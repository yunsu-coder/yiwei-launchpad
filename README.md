# 一苇 · YiWei Launchpad

> 一苇以航，轻量启程 — 个人综合启动页 / AI 工作台

## 功能模块

### 🏠 首页导航
- 书签分类管理（AI / 常用 / 开发 / 工具），支持自定义增删
- Bing 搜索、实时时钟、服务器状态监控
- 壁纸管理：URL 设壁纸、自动轮换、从文件/采集一键设置
- 主题色切换、深色模式

### 📁 文件中转站
- 拖拽上传、20GB 限额、进度条
- 文件预览：图片 / PDF / 视频 / 音频 / Markdown / JSON / 代码
- 批量多选删除、重命名、复制链接
- 图片文件一键设为首页壁纸

### 📝 Markdown 笔记
- 左右分屏实时预览，完整语法支持（表格、有序列表、任务列表、代码高亮）
- 30 秒自动保存、导出 PDF、导出到文件中转站
- 分类管理（全部 / 笔记 / 日记 / 技术 / 其他）

### 🔍 网页采集
- 图片 / 文本 / 视频 / 音频 四大类型爬取
- 深度渲染模式（Puppeteer）— 抓 JS 渲染页面
- 智能 URL 展开 + {起-止} 批量模式
- 缩略图预览、勾选选择性转存
- **B站视频** — 贴 BV 号或 b23.tv 短链即可下载 MP4
- **Internet Archive** — 搜索并下载背景音乐 MP3
- **Wallhaven** — 自动跟踪详情页获取 4K 原图
- **Pixabay** — 深度渲染抓取高清视频

### 📖 阅读器
- 支持 EPUB / PDF / TXT / Markdown
- 日间 / 护眼 / 夜间主题、字号可调、行高控制
- 自动滚屏、键盘翻页
- EPUB 目录导航
- 划词弹窗：一键保存笔记 / 翻译 / 问 AI

### 🌐 翻译
- 流式翻译（中文 / English / 日本語 / 한국어 / Français / Deutsch / Español / Русский / Português / العربية）
- 翻译历史记录、字幕文件支持 (.srt/.vtt)
- 一键复制、保存到笔记

### 🤖 AI 多模型 + 智能体
- 多模型并列对比（DeepSeek V4 / DeepSeek R1 / 通义千问 / Kimi / 豆包）
- **智能体模式**：6 个预设角色 + 系统提示词调教
  - 💻 代码助手 · ✍️ 创意写作 · 📚 学术顾问 · ⚖️ 法律顾问 · 🌐 翻译专家 · 🎨 图像生成
- 流式输出、会话管理、保存对话到文件

### 🎙️ 语音工具
- 文字转语音（6 种音色，Edge TTS）
- 语音转文字（浏览器 SpeechRecognition，支持中/英/日/韩）

### 🎵 全局音乐播放器
- 底部固定栏，不打断其他面板操作
- 播放列表、循环播放、随机播放、音量控制

## 跨模块连接

```
采集 → 文件 → 阅读器 → 翻译/AI → 语音
  │      │        │
  └──B站视频   设壁纸→首页    划词→笔记
```

共 20+ 条内部连接线，全程不离开页面。

## 启动

```bash
# 安装依赖
npm install

# 启动服务（默认 3000 端口）
npm start
# 或: systemctl --user restart dashboard
```

## 技术栈

Node.js · Vue 3 · Puppeteer · cheerio · sharp · epub.js · Nginx · Let's Encrypt · systemd
